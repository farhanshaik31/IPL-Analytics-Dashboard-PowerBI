# 🏏 IPL Analytics Dashboard (Power BI)

## 📌 Project Overview
This project analyzes Indian Premier League (IPL) match data using Power BI. The dashboard provides insights into team performance, match trends, player achievements, toss decisions, and venue statistics.

---

## 📊 Dashboard Features

- Total Matches
- Total Seasons
- Total Teams
- Team-wise Wins
- Matches by Season
- Top 10 Player of the Match Awards
- Toss Decision Analysis
- Interactive Filters (Season, City, Team, Venue)

---

## 🛠️ Tools Used

- Power BI
- Microsoft Excel / CSV
- SQL
- Python (EDA)

---

## 📂 Dataset

- matches.csv
- deliveries.csv 

---



---

## 🚀 Key Insights

- Mumbai Indians have won the highest number of matches.
- Most teams preferred to field after winning the toss.
- CH Gayle received the highest Player of the Match awards.
